# Virus-Outbreak-Management-System
A C++ console-based project using Data Structures and Algorithms (Struct, Linked List, Stack, Queue, Sorting, and Searching) to simulate a virus outbreak management system with patient tracking and testing management.
It is inspired by real-world epidemic response systems.


# The important features will be - 
1. Patient Tracking
2. Contact Tracing
3. Testing Management

# The main goals will be - 
1. To register and track patients
2. To manage the Testing Queue
3. Trace Contacts
4. Prioritize based on severity
5. Generate daily-based statistics
